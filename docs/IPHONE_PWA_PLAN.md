# iPhone / PWA Plan

## Cieľ
NBOS bude dostupný cez iPhone v prehliadači a neskôr ako PWA pridaná na plochu.

## MVP na iPhone
- mapa,
- zoznam objektov,
- detail objektu,
- pridať poznámku,
- nahrať fotku,
- uložiť GPS,
- vytvoriť úlohu,
- offline queue.

## Technický plán
1. Frontend ako PWA.
2. Service worker.
3. Local storage / IndexedDB.
4. Upload queue.
5. Sync endpoint.
6. Camera input.
7. Geolocation API.
